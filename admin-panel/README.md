# Tower Marketing Tracker - Admin Panel

A comprehensive React-based admin panel for managing the Tower/Pole Marketing Map System.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- ✅ Zone Management with Google Maps polygon drawing (Super Admin)
- ✅ User Management with role-based access (Super Admin)
- ✅ Pole Management with location selection and radius visualization
- ✅ Land Owner Management
- ✅ Line of Sight Calculator using Google Elevation API
- ✅ Settings Management (dynamic Google Maps API key)
- ✅ Role-based dashboards and statistics
- ✅ Zone-based access control for admins

## Tech Stack

React 18 | Vite | Material-UI | Google Maps API | React Router | Axios
